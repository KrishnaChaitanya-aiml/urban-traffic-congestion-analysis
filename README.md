# Urban Traffic Congestion Prediction with Adaptive Thresholding

## Overview
This project focuses on analyzing and predicting traffic congestion levels using statistical techniques. It classifies traffic into different congestion levels to support smart city traffic management.

## Problem Statement
Urban traffic congestion leads to delays, fuel wastage, and environmental issues. Identifying congestion levels helps optimize traffic flow.

## Approach
- Simulated traffic data using statistical distributions
- Applied adaptive thresholding based on mean and standard deviation
- Classified congestion into Low, Medium, and High levels
- Visualized traffic patterns and congestion zones

## Features
- Dynamic congestion classification
- Data visualization using plots
- Adaptive threshold-based decision making
